## Story
**As a** operador do Mr. Lion,
**I want** registrar o fornecedor e número da nota fiscal ao dar entrada no estoque,
**so that** eu mantenha rastreabilidade completa de cada lote recebido e possa consultar a origem de qualquer movimentação.

## Acceptance Criteria

1. O formulário de entrada de estoque (simples e em lote) tem campos opcionais: "Fornecedor" e "Nº NF"
2. O campo "Fornecedor" é um texto livre com sugestões (autocomplete) baseado em fornecedores já usados
3. O campo "Nº NF" aceita texto alfanumérico (ex: "NF-12345", "001234")
4. Ao confirmar entrada, a nota fiscal é incluída nas `notes` da movimentação no formato: `"Fornecedor: {nome} | NF: {numero} | {notas originais}"`
5. Na tabela de Relatórios, a coluna "Notas" exibe fornecedor e NF de forma destacada quando presentes
6. É possível filtrar o histórico por fornecedor no Relatórios (campo de busca de texto livre)
7. O autocomplete de fornecedores persiste localmente (localStorage) — não requer banco de dados
8. Os campos de NF são opcionais — não bloqueiam o registro se vazios

### buildMovementNotes
```typescript
export function buildMovementNotes(
  supplier?: string,
  invoiceNumber?: string,
  originalNotes?: string
): string {
  const parts: string[] = []
  if (supplier?.trim()) parts.push(`Fornecedor: ${supplier.trim()}`)
  if (invoiceNumber?.trim()) parts.push(`NF: ${invoiceNumber.trim()}`)
  if (originalNotes?.trim()) parts.push(originalNotes.trim())
  return parts.join(' | ')
}
```

### Persistência localStorage
```typescript
const STORAGE_KEY = 'mr-lion-suppliers'
const MAX_SUPPLIERS = 50

export function useSuppliers() {
  const [suppliers, setSuppliers] = useState<string[]>(() => {
    try {
      return JSON.parse(localStorage.getItem(STORAGE_KEY) ?? '[]')
    } catch { return [] }
  })

  const addSupplier = (name: string) => {
    if (!name.trim() || suppliers.includes(name)) return
    const updated = [name, ...suppliers].slice(0, MAX_SUPPLIERS)
    setSuppliers(updated)
    localStorage.setItem(STORAGE_KEY, JSON.stringify(updated))
  }

  return { suppliers, addSupplier }
}
```

### Destaque de fornecedor em notas
```typescript
function parseNotes(notes: string): { supplier?: string, invoice?: string, freeText?: string } {
  const supplierMatch = notes.match(/Fornecedor: ([^|]+)/)
  const invoiceMatch = notes.match(/NF: ([^|]+)/)
  const freeText = notes.replace(/Fornecedor: [^|]+\|?\s*/g, '').replace(/NF: [^|]+\|?\s*/g, '').trim()
  return {
    supplier: supplierMatch?.[1]?.trim(),
    invoice: invoiceMatch?.[1]?.trim(),
    freeText: freeText || undefined
  }
}
```
