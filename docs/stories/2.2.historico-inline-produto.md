## Story
**As a** gestor ou operador do Mr. Lion,
**I want** ver o histórico completo de movimentações de um produto clicando nele na lista de produtos,
**so that** eu investigue o histórico de um item específico sem precisar ir a Relatórios e configurar filtros manualmente.

## Acceptance Criteria

1. Ao clicar no nome de um produto na lista de Produtos, um drawer lateral se abre (desktop) ou bottom sheet (mobile)
2. O drawer exibe o nome do produto, SKU, categoria, estoque atual e mínimo no header
3. O drawer lista as movimentações do produto em ordem cronológica reversa (mais recente primeiro)
4. Cada linha de movimentação exibe: data/hora, tipo (badge colorido), quantidade, usuário e notas
5. O drawer carrega as últimas 50 movimentações e tem botão "Carregar mais" se houver mais
6. Um mini gráfico de linha mostra a evolução do estoque ao longo das últimas movimentações
7. O drawer tem botão de fechar (X) e fecha ao clicar fora (overlay)
8. O estado de loading exibe skeletons nas linhas da lista
9. O nome do produto na tabela/card tem cursor pointer e hover underline para indicar clicabilidade

### Calcular histórico de estoque retroativamente
```typescript
function calculateStockHistory(currentStock: number, movements: StockMovement[]): { date: string, stock: number }[] {
  // movements já vem em ordem reversa (mais recente primeiro)
  let stock = currentStock
  const history = [{ date: movements[0]?.created_at ?? new Date().toISOString(), stock }]

  for (const m of movements) {
    const delta = (m.action === 'in') ? -m.quantity : m.quantity // reverso para ir ao passado
    stock += delta
    history.push({ date: m.created_at, stock: Math.max(0, stock) })
  }

  return history.reverse() // cronológico
}
```
