## Story
**As a** gestor do Mr. Lion,
**I want** copiar o número do pedido com um clique e compartilhar o resumo diretamente no WhatsApp,
**so that** eu confirme pedidos com clientes de forma rápida sem digitar manualmente as informações.

## Acceptance Criteria

1. Cada pedido na lista exibe um número de identificação legível e curto (ex: `#MRL-0042`) baseado no índice/referência
2. Ao clicar no número do pedido, ele é copiado para o clipboard com feedback visual (ícone muda para ✓ por 2 segundos)
3. Cada pedido confirmado ou pendente tem botão "Compartilhar no WhatsApp" (ícone WhatsApp verde)
4. O botão WhatsApp abre `https://wa.me/?text=` com mensagem pré-formatada do pedido
5. A mensagem WhatsApp inclui: número do pedido, lista de itens (produto + quantidade), status e data
6. O campo "Referência" do formulário de novo pedido tem placeholder explicativo ("Ex: João Silva, mesa 5, pedido WA")
7. Pedidos entregues não exibem o botão WhatsApp (desnecessário após entrega)
8. Em mobile, o botão WhatsApp usa o app nativo do WhatsApp

```typescript
export function formatOrderNumber(id: string): string {
  return `#MRL-${id.slice(-4).toUpperCase()}`
}
```

### Hook useCopyToClipboard
```typescript
export function useCopyToClipboard() {
  const [copied, setCopied] = useState(false)
  const copy = async (text: string) => {
    await navigator.clipboard.writeText(text)
    setCopied(true)
    setTimeout(() => setCopied(false), 2000)
  }
  return { copied, copy }
}
```

### Mensagem WhatsApp
```typescript
export function generateWhatsAppMessage(order: DeliveryOrder): string {
  const items = order.items.map(i => `• ${i.product_name} × ${i.quantity}`).join('\n')
  const date = format(parseISO(order.created_at), "dd/MM/yyyy 'às' HH:mm", { locale: ptBR })
  const statusEmoji = order.status === 'confirmed' ? '✅' : '⏳'
  return encodeURIComponent(
    `🦁 *Pedido Mr. Lion*\nNº: ${formatOrderNumber(order.id)}\nData: ${date}\n\n*Itens:*\n${items}\n\nStatus: ${statusEmoji} ${order.status === 'confirmed' ? 'Confirmado' : 'Pendente'}`
  )
}
```
