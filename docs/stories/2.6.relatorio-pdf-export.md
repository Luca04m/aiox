## Story
**As a** gestor do Mr. Lion,
**I want** exportar relatórios em PDF com o logo e identidade visual da marca,
**so that** eu apresente relatórios profissionais para o CEO Ronaldo ou parceiros sem precisar de ferramentas externas.

## Acceptance Criteria

1. Na página de Relatórios, botão "Exportar PDF" gera e baixa o relatório como arquivo PDF
2. O PDF inclui header com logo do Mr. Lion (🦁), nome da empresa, e data de geração
3. O PDF inclui o período do filtro aplicado (De: XX/XX/XXXX Até: XX/XX/XXXX)
4. O PDF inclui o painel de resumo estatístico (totais por tipo)
5. O PDF inclui a tabela completa de movimentações (sem paginação — todos os registros filtrados)
6. As colunas do PDF são: Data/Hora, Produto, SKU, Tipo, Quantidade, Notas, Usuário
7. O PDF tem footer com número de página e "Gerado por Controle de Estoque Mr. Lion"
8. A tabela no PDF tem cores alternadas nas linhas (zebra) para legibilidade
9. O PDF usa cores da marca: navy escuro para headers, gold para acentos
10. A geração é feita no browser (client-side) sem enviar dados para servidores externos

### Estrutura do generateReportPDF
```typescript
import jsPDF from 'jspdf'
import 'jspdf-autotable'

export async function generateReportPDF(
  movements: StockMovement[],
  summary: MovementSummary,
  filters: { dateFrom?: string, dateTo?: string, productName?: string }
): Promise<void> {
  const doc = new jsPDF({ orientation: 'portrait', unit: 'mm', format: 'a4' })
  const pageWidth = doc.internal.pageSize.getWidth()

  // Header
  doc.setFontSize(20)
  doc.setTextColor(10, 11, 20) // navy
  doc.text('🦁 MR. LION — Relatório de Estoque', pageWidth / 2, 20, { align: 'center' })

  doc.setFontSize(10)
  doc.setTextColor(120, 120, 120)
  doc.text(`Gerado em: ${format(new Date(), "dd/MM/yyyy 'às' HH:mm")}`, pageWidth / 2, 28, { align: 'center' })

  // ... resto do conteúdo

  // Footer
  const pageCount = (doc.internal as any).getNumberOfPages()
  for (let i = 1; i <= pageCount; i++) {
    doc.setPage(i)
    doc.setFontSize(8)
    doc.text(`Página ${i} de ${pageCount}`, pageWidth - 20, doc.internal.pageSize.getHeight() - 10, { align: 'right' })
    doc.text('Gerado por Controle de Estoque Mr. Lion', 20, doc.internal.pageSize.getHeight() - 10)
  }

  doc.save(`relatorio-mr-lion-${format(new Date(), 'yyyy-MM-dd')}.pdf`)
}
```

### Tipos de movimento no PDF
- in → "Entrada" (texto verde hex)
- out → "Saída" (texto vermelho)
- adjustment → "Ajuste" (texto âmbar)
- loss → "Perda" (texto laranja)
