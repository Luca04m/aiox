## Story
**As a** operador do Mr. Lion usando o app no celular,
**I want** ver os dados em cards verticais ao invés de tabelas com scroll horizontal,
**so that** eu consiga visualizar e interagir com as informações sem precisar rolar para os lados.

## Acceptance Criteria

1. Em telas < 640px, a tabela de Produtos é substituída por uma lista de cards verticais
2. Cada card de produto exibe: nome, SKU, categoria (badge), estoque atual (com cor crítico/ok), estoque mínimo, status e ações
3. Em telas < 640px, a tabela de Relatórios é substituída por cards com as informações principais
4. Cada card de relatório exibe: data, produto, tipo (badge colorido), quantidade e notas
5. Em telas < 640px, os pedidos em OrdersPage já usam cards — verificar e garantir consistência visual
6. Cards de produto têm swipe area ou botão expandir para ver campos adicionais
7. Transição entre mobile card e desktop table é suave (apenas CSS, sem flash)
8. Performance: virtualização não é necessária para até 100 itens, mas não renderizar offscreen desnecessariamente

### Estratégia CSS-only (preferida — zero JS)
```tsx
{/* Desktop: tabela */}
<div className="hidden sm:block">
  <Table>...</Table>
</div>

{/* Mobile: cards */}
<div className="block sm:hidden space-y-3">
  {products.map(p => <ProductCard key={p.id} product={p} ... />)}
</div>
```

### Layout ProductCard
```tsx
<div className="rounded-lg border border-white/5 bg-secondary/50 p-4 space-y-3">
  {/* Header */}
  <div className="flex items-center justify-between">
    <span className="font-medium">{product.name}</span>
    <StatusBadge status={...} />
  </div>
  {/* Meta */}
  <div className="flex gap-3 text-sm text-muted-foreground">
    <span className="font-mono">{product.sku}</span>
    <CategoryBadge category={product.category} />
  </div>
  {/* Stock */}
  <StockBar current={product.current_stock} min={product.min_stock} />
  <div className="flex justify-between text-xs text-muted-foreground">
    <span>Atual: <span className={isCritical ? 'text-red-400' : 'text-foreground'}>{product.current_stock}</span></span>
    <span>Mínimo: {product.min_stock}</span>
  </div>
  {/* Actions (manager only) */}
  {isManager && (
    <div className="flex gap-2 pt-1">
      <Button size="sm" variant="outline" onClick={onEdit}>Editar</Button>
      <Button size="sm" variant="ghost" onClick={onToggle}>
        {product.active ? 'Desativar' : 'Ativar'}
      </Button>
    </div>
  )}
</div>
```
