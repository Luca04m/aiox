## Story
**As a** operador ou gestor do Mr. Lion,
**I want** ver mensagens claras e orientativas quando não há dados em telas e tabelas,
**so that** eu não fique confuso achando que o sistema está com erro quando uma lista está vazia.

## Acceptance Criteria

1. Todas as tabelas e listas com zero itens exibem um empty state com ícone, título e descrição contextual
2. O empty state de Produtos sugere "Cadastre o primeiro produto" com botão de ação (manager only)
3. O empty state de Pedidos sugere "Nenhum pedido em andamento" com botão "Novo Pedido"
4. O empty state de Relatórios/Histórico diz "Nenhuma movimentação encontrada para os filtros aplicados"
5. O empty state do Dashboard (sem movimentações recentes) exibe mensagem de boas-vindas com orientação
6. Ícones dos empty states usam `lucide-react` e são temáticos por contexto
7. Empty states são responsivos e estilizados conforme o design system (dark navy + gold)
8. Quando filtros estão ativos e resultam em zero itens, o empty state indica "Tente ajustar os filtros"
